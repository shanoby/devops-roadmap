# Задание 1: Сетевая конфигурация

## 🎯 Цель
Создать сеть с 3 серверами (web, db, cache) с изоляцией.

## 📋 Задачи

### 1. Создание сети
- [ ] Создать VPC с публичным и приватным сабнетом
- [ ] Настроить route tables
- [ ] Создать security groups

### 2. Развертывание серверов
- [ ] Web сервер в публичном сабнете
- [ ] DB сервер в приватном сабнете
- [ ] Cache сервер в приватном сабнете

### 3. Настройка firewall
- [ ] SSH доступ к web серверу
- [ ] HTTP/HTTPS доступ к web серверу
- [ ] DB доступ только с web сервера
- [ ] Cache доступ только с web сервера

## ✅ Проверка
- [ ] Web сервер доступен из интернета
- [ ] DB сервер недоступен из интернета
- [ ] Cache сервер недоступен из интернета
- [ ] Web может подключиться к DB
- [ ] Web может подключиться к Cache
