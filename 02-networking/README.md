# 2. Сети и протоколы

## 🎯 Что изучить

### Модели
- OSI (7 уровней)
- TCP/IP стек

### Протоколы
- HTTP/HTTPS (методы, статус коды, заголовки)
- TCP vs UDP (соединения, надежность)
- DNS (записи A, AAAA, CNAME, MX)
- SSH (ключи, tunneling)
- FTP/SFTP

### Инструменты
- tcpdump, wireshark (анализ трафика)
- nmap (сканирование портов)
- netstat/ss (сетевые соединения)
- ipcalc (вычисление сетей)

### Сетевые устройства
- Маршрутизаторы
- Коммутаторы
- Firewall
