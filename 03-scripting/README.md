# 3. Скриптинг

## 🎯 Что изучить

### Bash
- **Переменные** - объявление, типы, подстановка
- **Циклы** - for, while, until, break, continue
- **Условия** - if, case, test, [[]]
- **Функции** - аргументы, возвращаемые значения, scope
- **Работа с файлами** - redirection, pipes, globbing
- **Регулярные выражения** - grep, sed, awk
- **Обработка ошибок** - exit codes, trap, set -e

### Python
- **requests** - HTTP запросы, сессии, аутентификация
- **subprocess** - выполнение команд, PIPE, timeout
- **paramiko** - SSH подключения, SFTP, ключи
- **argparse** - парсинг аргументов командной строки
- **logging** - уровни логов, handlers, formatters
- **Автоматизация** - скрипты для развертывания, конфигурации
