# 4. Git и VCS

## 🎯 Что изучить

### База
- **commit** - создание коммитов, сообщения, amend
- **push, pull, fetch** - работа с удаленным репозиторием
- **branch, merge, rebase** - ветвление, слияние, перебазирование
- **cherry-pick** - выборочное применение коммитов
- **reset, revert, checkout** - отмена изменений

### Workflow
- **Git Flow** - master, develop, feature, release, hotfix
- **GitHub Flow** - простой workflow с pull request
- **Trunk-based development** - короткоживущие ветки

### Инструменты
- **pre-commit hooks** - проверка кода перед коммитом
- **git-lfs** - работа с большими файлами
- **.gitignore** - исключение файлов
- **Git tags** - версионирование
