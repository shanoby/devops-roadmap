# 5. CI/CD

## 🎯 Что изучить

### Jenkins
- **Jenkinsfile** - Declarative Pipeline, Scripted Pipeline
- **Плагины** - Docker, Git, Kubernetes, Credentials
- **Агенты** - static, dynamic, Docker, Kubernetes
- **Интеграция с Git** - webhook, polling, credentials

### GitHub Actions
- **Workflows** - workflow syntax, triggers, jobs, steps
- **Actions marketplace** - готовые actions, создание своих
- **Runners** - hosted runners, self-hosted runners
- **Secrets и variables** - защита секретов, переменные окружения

### GitLab CI
- **.gitlab-ci.yml** - stages, jobs, variables
- **Runners** - shared, specific, Docker runners
- **Variables, secrets** - CI/CD variables, protected variables
- **Artifacts и cache** - кэширование, артефакты

## 📖 Документация
- [Jenkins Handbook](https://www.jenkins.io/doc/book/) - официальная документация
- [GitHub Actions](https://docs.github.com/en/actions) - официальная документация
- [GitLab CI/CD](https://docs.gitlab.com/ee/ci/) - официальная документация
- [GitHub Actions Marketplace](https://github.com/marketplace?type=actions) - готовые actions
