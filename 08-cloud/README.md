# 8. Облачные провайдеры

## 🎯 Что изучить

### AWS
- **EC2** - instances, AMIs, security groups, key pairs
- **S3** - buckets, objects, versioning, lifecycle, ACL
- **VPC** - subnets, route tables, internet gateway, NAT gateway
- **IAM** - users, groups, roles, policies, permissions
- **EKS, ECS** - Kubernetes, Fargate, EC2 launch type
- **CloudWatch, CloudTrail** - мониторинг, аудит

### Azure
- **VM** - virtual machines, extensions, availability sets
- **Storage** - blob, file, queue, table storage
- **VNet, NSG** - виртуальная сеть, группы безопасности
- **AKS** - Azure Kubernetes Service, node pools
- **Azure Monitor, Key Vault** - мониторинг, секреты

### GCP
- **Compute Engine** - VM instances, images, snapshots
- **Cloud Storage** - buckets, objects, IAM
- **VPC** - сети, firewall rules, routes
- **GKE** - Google Kubernetes Engine, clusters, node pools
- **Cloud Monitoring, Secret Manager** - мониторинг, секреты
